# Laravel Warehouse & Task Manager

A warehouse and task management system built with Laravel as a university course project.

## Installation

```bash
git clone [your-repository-url]
cd warehouse-manager
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve
```

## Requirements

- PHP >= 8.0
- Composer
- MySQL/PostgreSQL
- Laravel 10.x
